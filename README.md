Задание 4. Генератор статических сайтов.

Используя любой генератор создать статический сайт.
Необходимо уметь:
Cоздавать простые страницы, составные страницы(блог/новостная лента/т.д.)
Настраивать элементы дизайна и шаблоны

1) https://gohugo.io/
